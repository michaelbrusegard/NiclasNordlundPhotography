nature.html, animals.html, architectural.html, portrait.html, wedding.html, sport.html Create display of the showcase pictures on their sites. Bigger pictures on landscape etc. Displayed after image width. Fade in animation for the photos. Magnus

showcase images Get the files for showcase images from Niclas. Michael
showcase.css Styling for showcase pages. Magnus

home.html Homepage with information about Niclas, quotes and best pictures. Also the footer with contact info easily accesible. All
shop.html Create the checkout system with shopping display and checkout cart. Store prices and send to Stripe for checkout. (If there is time implement a webhook to send the bought images to the customer). Martin and Michael
shop.css Shop styling. All
checkoutSystem.js System for tracking which images are added to the shopping cart ready for purchase. This information should be sent to the payment solution on Stripe. Martin
quotesTyping.js Text being "written" on the page. Magnus
splashOnClick.js Splash effect on nav bar when selecting showcase pages. Erik
home.css Homepage styling and generic styling. All
shoppingCart.js Red dot to indicate items in cart, and an animation from the item being dropped in the cart. Erik
slideTransition.js Animation when going from shopping -> showcase or showcase -> shopping. Martin and Michael
Final check Double check all files, make sure they are valid, no broken links, etc. All
